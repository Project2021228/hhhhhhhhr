Решение уравнений связанных с системами счисления.

1. Решите уравнение 425 + x = 11223.Ответ запишите в четверичной системе счисления. Основание системы счисления указывать не нужно.

  • reorder
    Пример решения:
    Переведём числа в десятичную систему счисления:
    425 = 4 * 51 + 2 * 50 = 20 + 1 = 2110.
    11223 = 1 * 33 + 1 * 32 + 2 * 31 + 2 * 30 = 27 + 9 + 6 + 2 = 4410.
    Теперь решим уравнение в десятичной системе счисления:
    21 + x = 44
    x = 44 - 21 = 23.
    23 = 1134.

    Ответ: 113.


2. Решите уравнение 224x + 1 = 1018. Ответ запишите в десятичной системе счисления

  • reorder
    Пример решения:
    Перенесём всё что нам известно в правую сторону. Получаем:
    224x = 1008.
    Переведём 1008 и 224x в десятичную систему счисления:
    1008 = 1 * 82 + 0 * 81 + 0 * 80 = 64 + 0 + 0 = 6410.
    224x = 2 * x2 + 2 * x1 + 4 * x0 = (2x2 + 2x + 4)10
    Получаем, что 2x2 + 2x + 4 = 64.
    Сокращаем обе стороны уравнения на 2 и получаем: x2 + x - 30 = 0.
    Запишем уравнение в виде разности: x2 + 6x - 5x - 30 = 0.
    Вынесем общим вножитель за скобки: x(x + 6) - 5(x + 6) = 0.
    Получается, что x равен либо (x + 6 = 0) либо (x - 5) = 0.
    x = -6 или x = 5.
    Основание системы счиления не может быть отрицательным, поэтому x равен 5.

    Ответ: x = 5.


3. Решите уравнение 44x+5 - 445 = 5210. Ответ запишите в десятичной системе счисления.

  • reorder
    Пример решения:
    Для удобства переведём всё в десятичную систему счисления:
    445 = 4 * 51 + 4 * 50 = 20 + 4 = 2410.
    44x+5 = 4 * (x+5)1 + 4 * (x+5)0 = 4 * (x+5) + 4 = 4x + 20 + 4 = (4x + 24)10.
    Получаем: 4x + 24 - 24 = 52.
    Переносим всё, что нам известно в правую сторону, а x оставляем в левой части уравнения.
    Получаем: 4x = 52 + 24 - 24; 4x = 52.
    x = 52/4. x = 13.

    Ответ: x = 13.


4. Решите уравнение 101x + 1310 = 101x+1. Ответ запишите в десятичной системе счисления.

  • reorder
    Пример решения:
    Для удобства переведём всё в десятичную систему счисления:
    101x = 1 * x2 + 0 * x1 + 1 * x0 = x2 + x + 1.
    101x+1 = 1 * (x+1)2 + 0 * (x+1)1 + 1 * (x+1)0 = (x+1)2 + 1 = x2 + 2x + 2.
    Получаем уравнение: x2 + x + 1 + 13 = x2 + 2x + 2.
    Переносим всё неизвестное вправо, а известное влево: x2 + x - (x2 + 2x) = 2 - (1 + 13).
    После сокращения уравнение приобретает следуйщий вид: -x = -12.
    Домоножаем обе стороны на (-1) и получаем, что x = 12.

    Ответ: x = 12.


5. Решите уравнение 104x + 20x = 8410. Ответ запишите в двоичной системе счисления.

  • reorder
    Пример решения:
    Переведём все числа в десятичную систему счисления:
    104x = 1 * x2 + 0 * x1 + 4 * x0 = (x2 + 4)10.
    20x = 2 * x1 + 0 * x0 = 2x10.
    Получаем уравнение: x2 + 4 + 2x = 84.
    Переносим всё в левую часть уравнения: x2 + 2x + 4 - 84 = 0; x2 + 2x - 80 = 0.
    Решаем уравнение с помощью дискриминанта: D = b2 - 4ac;
    D = 22 - 4 * 1 * (-80) = 4 + 320 = 324. √324 = 18.
    Так как дискриминант положительный, значит имеется два корня.
    Получаем, что x1 = 8, а x2 = -10. Основание системы счисления - это натуральное число, следовательно
    решением будет являться x = 8.

    Ответ: x = 8.

Алгоритмы переводов систем счисления

Перевод из десятичной системы счисления в любую другую
Чтобы перевести десятичное число в систему счисления с другим основанием, нужно это число разделить на основание.
Полученное частное снова разделить на основание, и дальше до тех пор, пока частное не окажется меньше основания. В результате записать в одну строку последнее частное и все остатки, начиная с последнего.
Например: 1) переведем десятичное число 46 в двоичную систему счисления. Ответ: 46 = 1011102
2) переведем десятичное число 672 в восьмеричную систему счисления. Ответ: 672 = 12408
3) переведем десятичное число 934 в шестнадцатеричную систему счисления. Ответ: 934 = 3A616
Перевод из любой системы счисления в десятичную
Чтобы выполнить перевод целого числа из любой позиционной системы счисления в десятичную, нужно представить число в виде суммы разрядных слагаемых. Цифры и разрядные единицы записываются в десятичной системе счисления.
Например: переведем двоично число 101102 в десятичную систему счисления. Ответ: 101102 = 22

101102 = 1 • 24 + 0 • 23 + 1 • 22 + 1 • 21 + 0 • 20 = 16 + 4 + 2 = 22
